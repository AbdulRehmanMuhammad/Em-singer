Kindly run "emSigner_unix_1_7_0.sh" file with below command to install the emSigner services.

command to provide access priviledges: chmod 777 emSigner_unix_1_7_0.sh
command to install the services: ./emSigner_unix_1_7_0.sh

After installation go to the installed directory and use below command to provide access priviledges to emSigner services.

command: chmod -R 777 emSigner  (here emSigner is the installed root directory for services)

After that double click everytime on "startserver" file to start emSigner server.
